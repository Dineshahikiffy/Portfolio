My Project

Snake Game   Github

This Snake game is a classic arcade game where the player controls a snake that moves around the screen, eating food to grow longer. The goal is to avoid colliding with the walls of the game area or with the snake's own body. As the snake eats food, it grows longer, making it more challenging to navigate the game space.

Technologies used: Python, Tkinter

Tic Toc Toe App   Github

This Tic-Tac-Toe game is a simple yet engaging implementation using HTML, CSS, and JavaScript. It provides a classic two-player experience, where players take turns to mark X or O on a 3x3 grid. The game checks for win conditions after each move and declares a winner or a draw accordingly. It's a fun and interactive way to enjoy this timeless game right in your browser.

Technologies used: HTML, CSS, JavaScript, CodePen, Tiny.host, Web to App

Flight Ticket Booking System   Github

A console-based application implemented in Python, the flight ticket booking system enables users to search for flights based on their preferences, including departure city, destination, and date. Users can select a flight and book tickets for themselves and others, streamlining the booking process.

Technologies used: Python
